Complete template syntax reference

The text before the first section is the learner-facing question. Insert generated values with {VARIABLE}. Mathematical templates use Definitions and Formula; fixed semantic templates may use only Metadata and Semantic Answer.

Supported sections

## Metadata## Definitions## Mappings## Formula## Constraints## Answer## Answers## Semantic Answer## Choices## Feedback

Metadata terms

TITLE:SUBJECT:TOPIC:TYPE:DIFFICULTY:SEED:TAGS:LANGUAGE:MAX_CONSTRAINT_ATTEMPTS:

SEED accepts random or an integer from 0 to 4294967295.

VALUE: description (600)Fixed integer, decimal, or text
VALUE: description (5..8)Inclusive integer range
VALUE: description (1.0..10.0; step=0.5)Stepped decimal range
VALUE: description (even, odd, none)Numeric or text set
MAPPED_VALUE: SOURCE_VALUEMapping header followed by key=value lines
FRAME_BITS = 1 + DATA_BITS + PARITY_BITS + STOPDependent formula variable
ANSWER = ...Conventional final answer variable
DATA_BITS > 0Constraint; supports ==, !=, <, <=, >, >=, AND, OR, NOT

Answer terms

VALUE:LABEL:TYPE:UNIT:ROUND:TOLERANCE:TOLERANCE_TYPE:EQUIVALENCE:ACCEPT:

Use ## Answers with one variable block per answer when a problem requires multiple results.

Semantic Answer terms

REFERENCE:STRICTNESS:ESSENTIAL_CONCEPTS:SUPPORTING_CONCEPTS:ACCEPTED_EXPRESSIONS:KNOWN_INCORRECT_CLAIMS:

Semantic templates can be instantiated without numeric validation. Gemini is required only for grading the learner response.

Choices terms

CORRECT:DISTRACTOR:SHUFFLE:

The section name and terms are reserved for multiple-choice template expansion. Deterministic templates currently generate single-answer or multi-answer free-response instances.

Feedback terms

HINT:SOLUTION:EXPLANATION:

Feedback text may include generated or calculated placeholders such as {FRAME_BITS}.

Formula support

Operators: + - * / % ^ ( ). Constants: PI, E. Functions: abs, round, floor, ceil, min, max, sqrt, pow.
